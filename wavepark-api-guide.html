OVERVIEW

웨이브파크 예약 API

wavepark.co.kr의 예약 시스템에서 사용하는 내부 API입니다. 날짜를 기준으로 예약 가능한 상품 목록을 조회할 수 있으며, 응답은 렌더링된 HTML 형태로 반환됩니다.

ℹ️
Base URL: https://www.wavepark.co.kr
응답 Content-Type은 application/json이나, html 필드에 렌더링된 HTML이 담깁니다.
ENDPOINT 01

날짜별 상품 목록 조회

특정 날짜에 예약 가능한 서핑 상품(레슨, 자유서핑, 패키지) 목록을 조회합니다.

POST /bookingone/getList

REQUEST

📋
Content-Type: application/x-www-form-urlencoded
파라미터 타입 필수 설명 / 예시

조회 연도
예) "2026"

조회 월 (1~12, 앞에 0 없이)

조회할 특정 날짜 (yyyy-MM-dd 형식)
예) "2026-04-25"

RESPONSE

응답 구조

응답은 JSON 객체이며, 렌더링된 HTML이 html 키에 담겨 옵니다.

string 에러 메시지. 정상 응답 시 빈 문자열 ""

string (HTML) 선택한 날짜에 해당하는 상품 카드들의 HTML 마크업. <article class="card"> 요소의 연속.

data 속성 상세

data-pickdate
예약 대상 날짜
"2026-04-26"
data-itemidx
상품 고유 ID (예약 API 호출 시 사용 추정)
"30902", "27812", "18"
data-sectype
상품 카테고리 코드 (하단 코드표 참고)
"30", "70", "90"
card-badge-origin
상품 유형 텍스트
단품 패키지

PARSING GUIDE

HTML 파싱 방법

응답 HTML에서 상품 정보를 구조화된 데이터로 추출하는 방법입니다.

KOTLIN · Jsoup 파싱 
// build.gradle: implementation("org.jsoup:jsoup:1.17.2")

data class WaveparkItem(
    val pickDate: String,
    val itemIdx: String,
    val secType: String,
    val badgeType: String,   // "단품" | "패키지"
    val name: String,
    val price: String,
)

fun parseItems(html: String): List<WaveparkItem> {
    val doc = Jsoup.parseBodyFragment(html)
    return doc.select("article.card").map { el ->
        WaveparkItem(
            pickDate = el.attr("data-pickdate"),
            itemIdx  = el.attr("data-itemidx"),
            secType  = el.attr("data-sectype"),
            badgeType = el.selectFirst(".card-badge-origin")?.text() ?: "",
            name     = el.selectFirst(".card-name")?.text() ?: "",
            price    = el.selectFirst(".card-price")?.text() ?: "",
        )
    }
}
PYTHON · BeautifulSoup 
from bs4 import BeautifulSoup

def parse_items(html: str) -> list[dict]:
    soup = BeautifulSoup(html, "html.parser")
    items = []
    for card in soup.find_all("article", class_="card"):
        items.append({
            "pickDate" : card.get("data-pickdate"),
            "itemIdx"  : card.get("data-itemidx"),
            "secType"  : card.get("data-sectype"),
            "badgeType": card.find(class_="card-badge-origin").text.strip(),
            "name"     : card.find(class_="card-name").text.strip(),
            "price"    : card.find(class_="card-price").text.strip(),
        })
    return items

KOTLIN · Jsoup 파싱 
data class TimeSlot(
    val pickDatetime: String,  // "2026-04-25 10:00:00"
    val itemIdx: String,
    val label: String,         // "10:00 ~ 12:00"
    val remaining: Int,        // 11
    val capacity: Int,         // 15
)

data class ItemDetail(
    val name: String,
    val slots: List<TimeSlot>,
)

fun parseDetail(html: String): ItemDetail {
    val doc = Jsoup.parseBodyFragment(html)
    val name = doc.selectFirst(".item-name")?.text() ?: ""

    val slots = doc.select("button.btnTimeCheck").map { btn ->
        val cap  = btn.selectFirst("span.opt-cap")?.text() ?: "0/0"
        val parts = cap.split("/").map { it.trim().toIntOrNull() ?: 0 }
        TimeSlot(
            pickDatetime = btn.attr("data-pickdatetime"),
            itemIdx      = btn.attr("data-itemidx"),
            label        = btn.ownText().trim(),
            remaining    = parts.getOrElse(0) { 0 },
            capacity     = parts.getOrElse(1) { 0 },
        )
    }
    return ItemDetail(name, slots)
}

ENDPOINT 03

파도위치 · 서핑구역 조회

타임슬롯 선택 후 호출되는 API입니다. 파도위치(좌측/우측)와 각 위치별 서핑구역 버튼 HTML을 반환합니다. getDetail의 waveArea_0 / zoneArea_0 영역을 이 응답으로 채웁니다.

🔄
호출 순서: getList → getDetail → getWaveZone (타임슬롯 클릭 시)

핵심 추출 필드

data-schidx
스케줄 고유 ID — waveHtml과 zoneHtml을 연결하는 키. 파도위치 버튼의 schidx = 해당 구역 버튼의 schidx
좌측: 44997 / 우측: 44996
data-wave
파도위치 번호 (추정: 좌=1, 우=2)
"1" → 좌측, "2" → 우측
data-maxcnt
해당 구역 최대 인원 — 좌/우 정원이 다를 수 있음
좌측: 20명, 우측: 4명
data-zone
구역 코드
"20" (중급리프)
⚠️
좌/우 정원이 다릅니다: 같은 타임슬롯이라도 좌측(maxcnt=20)과 우측(maxcnt=4)의 수용 인원이 다릅니다. 잔여석 표시 시 구역별로 따로 처리해야 합니다. 현재 응답에 현재 예약된 인원(점유수)은 포함되지 않으므로 별도 확인이 필요할 수 있습니다.

data class WavePosition(
    val schIdx: String,    // "44997"
    val label: String,     // "좌측" | "우측"
)

data class SurfZone(
    val schIdx: String,    // 연결된 파도위치 schIdx
    val wave: String,      // "1"=좌측, "2"=우측 (추정)
    val zone: String,      // 구역 코드 "20"
    val maxCnt: Int,       // 최대 정원
    val itemTime: String,  // "2026-04-25 12:00:00"
    val label: String,     // "중급리프"
)

data class WaveZoneResult(
    val positions: List<WavePosition>,
    val zones: List<SurfZone>,
)

fun parseWaveZone(json: JSONObject): WaveZoneResult {
    val waveDoc = Jsoup.parseBodyFragment(json.getString("waveHtml"))
    val zoneDoc = Jsoup.parseBodyFragment(json.getString("zoneHtml"))

    val positions = waveDoc.select("button.btnWaveCheck").map {
        WavePosition(
            schIdx = it.attr("data-schidx"),
            label  = it.text(),
        )
    }

    val zones = zoneDoc.select("button.btnWaveZoneCheck").map {
        SurfZone(
            schIdx   = it.attr("data-schidx"),
            wave     = it.attr("data-wave"),
            zone     = it.attr("data-zone"),
            maxCnt   = it.attr("data-maxcnt").toIntOrNull() ?: 0,
            itemTime = it.attr("data-itemtime"),
            label    = it.text(),
        )
    }

    return WaveZoneResult(positions, zones)
}

// 파도위치 선택 후 해당 구역만 필터링
fun WaveZoneResult.zonesFor(schIdx: String) =
    zones.filter { it.schIdx == schIdx }

// Coroutine + OkHttp + Jsoup
suspend fun fetchWaveparkItems(date: LocalDate): List<WaveparkItem> {
    return withContext(Dispatchers.IO) {
        val body = FormBody.Builder()
            .add("yy",        date.year.toString())
            .add("mm",        date.monthValue.toString())
            .add("selectYmd", date.toString())   // yyyy-MM-dd
            .build()

        val request = Request.Builder()
            .url("https://www.wavepark.co.kr/bookingone/getList")
            .post(body)
            .build()

        val responseJson = OkHttpClient()
            .newCall(request).execute()
            .body?.string() ?: return@withContext emptyList()

        val html = JSONObject(responseJson).getString("html")
        parseItems(html)
    }
}
